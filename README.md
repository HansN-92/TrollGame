# Troll Game 1: The Trollening

A browser-based combat RPG built with vanilla JavaScript, HTML, and CSS.

## About

Fight your way through increasingly difficult troll encounters across multiple stages.
Manage your stamina, weapons, and inventory carefully — the troll hits back hard,
regenerates health when wounded, and unleashes a special attack when cornered.

Between stages you can visit the shop to gear up, or head to camp to forage,
hunt, cook, brew potions, and rest before the next fight.

## Features

- ⚔️ Turn-based combat with stamina management
- 🪓 Weapon pickups (Hammer, Axe, Chainsaw) with varying strength and stamina cost
- 🧪 Inventory system — berries, meat, big meals, and HP potions
- 🏕️ Camp system — forage, hunt, cook, and brew potions with limited actions
- 🛒 Shop with permanent upgrades (Ring of Strength, Max HP Armor, Boots of Speed)
- 📈 Stage scaling — troll grows stronger each round
- 🔄 Full restart support
